# TKT-0216: 食材カードの期限・換算バッジと選択チェック配置を整理

## 結論

食材カードの食材名行から期限バッジと換算バッジを外し、食材名が見切れにくい配置へ変更した。

## 変更内容

- 食材名行は食材名だけを表示するようにした。
- `期限切` / `あと何日` バッジは、保存場所・購入日の下へ移動した。
- `1パック=xxg` などの換算バッジは、数量ステッパーの右側へ移動した。
- 数量ステッパーは横幅を取りすぎないよう、カード内でコンパクトな幅にした。
- 選択チェックボックスはカード上部の左端寄りへ移動した。
- 期限バッジ行と換算バッジ欄は、バッジが無いカードでも空エリアを予約するようにした。
- 数量行は左のステッパー欄と右の換算バッジ欄を固定的な2カラムにし、`1パック=xxg` の有無でカードの横位置が崩れないようにした。
