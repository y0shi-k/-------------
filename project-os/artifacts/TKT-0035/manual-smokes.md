# Manual Smokes

## Canvas 環境での手動確認項目

### 1. レシピ選択モーダルのソート

1. 献立・レシピ → スケジュールタブ → 任意の日付枠の「＋」をタップ
2. レシピ選択モーダルが開く
3. 「名前」「追加日」「前回調理」「調理回数」のソートボタンが検索欄の下に表示されることを確認
4. 「調理回数」をタップ → 昇順/降順が切り替わることを確認
5. 並び順が変わることを確認

### 2. レシピ選択モーダルの材料名検索

1. レシピ選択モーダルを開く
2. 検索欄に材料名（例: 「卵」「醤油」）を入力
3. 材料名を含むレシピが検索結果に表示されることを確認
4. レシピ名にその文字列が含まれなくても、材料に含まれるレシピがヒットすることを確認

### 3. レシピ詳細からスケジュール追加

1. 献立・レシピ → レシピ集タブ → 任意のレシピをタップ
2. 「スケジュールに追加」ボタンが表示されることを確認
3. 「スケジュールに追加」をタップ
4. ミニカレンダーモーダルが開くことを確認
5. カレンダー横スクロールで30日分表示されることを確認
6. 日付をタップ → 朝・昼・夜のボタンが展開されることを確認
7. 「夜」をタップ → 献立が追加され、モーダルが閉じることを確認
8. 献立タブに追加されたレシピが表示されることを確認
