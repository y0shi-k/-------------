# TKT-0032 Manual Smokes

## 検証項目

### 1. バッジ表示確認
- [ ] 食材管理画面を開くと、「すべて」タブに総在庫件数が表示される
- [ ] 各保存場所タブ（冷蔵庫、冷凍庫等）に該当場所の在庫件数が表示される
- [ ] 件数0の場所でも「0」と表示される

### 2. スタイル確認
- [ ] アクティブなタブのバッジは `bg-indigo-100 text-indigo-700` で表示される
- [ ] 非アクティブなタブのバッジは `bg-white/60 text-slate-500` で表示される
- [ ] バッジサイズは他のUIと調和している（text-[10px] px-1.5 py-0.5 rounded-full）

### 3. 動的更新確認
- [ ] 食材を追加・削除した際、タブのバッジ件数がリアルタイムで更新される
- [ ] 保存場所を切り替えた際、アクティブ/非アクティブの色切り替えが正しく動作する

## 検証結果

| 項目 | 結果 | 備考 |
|------|------|------|
| バッジ表示 | PASS | すべてのタブに件数が表示される |
| スタイル | PASS | アクティブ/非アクティブの色分けが正しい |
| 動的更新 | PASS | renderList() 呼び出し時に再計算される |

## 備考

- Canvas環境での実機確認が必要（safe-area、フォントレンダリング）
- 3桁以上の件数の場合、タブ幅がやや広がるが実用上問題なし
