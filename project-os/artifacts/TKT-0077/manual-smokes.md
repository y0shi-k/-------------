# Manual Smokes

Gemini Canvas 実表示はユーザー確認対象。

## Checklist

- レシピ削除で確認モーダルが表示され、キャンセルすると削除されない。
- レシピ削除で「削除する」を押すとUIから消え、同期バーに未同期件数が出る。
- 在庫単体削除で確認モーダルが表示され、キャンセル/確定が分岐する。
- 在庫一括削除で確認モーダルが表示され、キャンセル/確定が分岐する。
- 在庫数量を `-` で0にする時、削除確認モーダルが表示される。
- 買い物リスト単体削除で確認モーダルが表示され、キャンセル/確定が分岐する。
- 買い物リスト一括削除で確認モーダルが表示され、キャンセル/確定が分岐する。
- 献立一括削除は未完了のみの選択でも確認モーダルが表示される。
