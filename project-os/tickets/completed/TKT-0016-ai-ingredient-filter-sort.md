# TKT-0016: AI食材選択モーダル拡張（検索・ソート・フィルタ・ジャンル分類）

## 目的
AI献立・レシピの「指定食材から」で食材選択がしやすくなるよう、検索・ソート・保存場所フィルタ・ジャンルフィルタを追加する。

## Acceptance Criteria
- [ ] 在庫登録・編集フォームの分類 `<select>` に「野菜」「肉」「魚」「乳製品」「穀物」「その他」が追加されている
- [ ] 食材選択モーダル上部に検索欄があり、食材名でリアルタイム絞り込みできる
- [ ] 食材選択モーダルで「名前」「購入日」「期限」でソートでき、昇降切り替えできる
- [ ] 食材選択モーダルで保存場所でフィルタできる
- [ ] 食材選択モーダルでジャンル（分類）でフィルタできる
- [ ] 既存の期限色分け（赤・黄）がそのまま表示される
- [ ] 在庫管理側のソート状態と干渉しない
- [ ] verify が通る

## Required Evals
- `verify` コマンド成功
- `project-os/artifacts/TKT-0016/report.md` に完了報告を残す

- `app.html`
