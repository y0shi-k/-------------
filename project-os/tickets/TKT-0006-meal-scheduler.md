## 目的

モードBに「献立スケジュール」機能を実装する。週間（7日間）の縦スクロールリストビューで、各日付に朝・昼・晩のスロットを持たせ、レシピ集からの手動割り当て・削除を可能にする。

## タスク

- [ ] `handleInit` のGASペイロードに `献立スケジュール` シートのヘッダー追加処理を追加
- [ ] モードB内に「献立スケジュール」サブタブ/セクションUIを追加
- [ ] 週間リストビューUIを実装
  - [ ] 「前の週」「今週」「次の週」ボタン
  - [ ] 7日分の日付カード（縦スクロール）
  - [ ] 各日付カード内に朝・昼・晩のスロット
- [ ] 空スロットに「＋追加」ボタンを表示し、タップでレシピ選択モーダルを開く
- [ ] レシピ選択モーダルを実装（レシピ集一覧＋検索）
- [ ] 割り当てフローを実装（GAS経由で `献立スケジュール` シートへ保存）
- [ ] 割り当て済みスロットのタップで「別レシピに変更」「削除」メニューを表示
- [ ] 削除/変更フローを実装（GAS経由でシート更新・削除）
- [ ] `state.schedule` を追加し、週切り替え時に該当期間のデータをGASから取得
- [ ] verify コマンド実行
- [ ] artifacts を `project-os/artifacts/TKT-0006/` に作成

## Acceptance

- モードBの「献立スケジュール」で今週の7日間が縦スクロールで表示される
- 各日付カードに朝・昼・晩のスロットがあり、空なら「＋追加」が表示される
- 「＋追加」でレシピ選択モーダルが開き、レシピを選んで割り当てられる
- 割り当て後、スロットにレシピ名が表示される
- 割り当て済みスロットをタップして「削除」を選ぶと消える
- 「前の週」「次の週」で週が切り替わり、該当期間の献立が表示される
- HTML構文チェックが通る
- `executeGAS` と `GAS_URL` が残っている
