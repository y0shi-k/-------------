# Summary

レシピ一覧カードのジャンル要約を固定3件表示から、カード内の残り幅に応じた可変表示へ変更する。省略分の `+N` はブラウザ標準 `title` ではなく独自ツールチップで即時表示する。

## 仕様

- 一覧カード描画時、ジャンル要約コンテナに全ジャンル配列を保持する。
- 描画後にコンテナの実幅を測定し、全件表示から順に減らして収まる最大件数を採用する。
- 省略がある場合のみ `+N` チップを表示する。
- `+N` チップは `mouseenter` で即時ツールチップを表示し、`mouseleave` で閉じる。
- `resize` 時は debounce して再計算する。

## 非対象

- ジャンル保存形式変更
- レシピ一覧検索仕様変更
- GAS/Spreadsheet変更
