# Summary

画面下部に出ていた主要操作ボタンを、対象画面の上部操作領域へ移動して、リスト操作中の視線移動とスクロール量を減らす。処理そのものは既存関数を使い、同期・GAS・データ構造は変更しない。

## 仕様

- レシピ詳細モーダルの「スケジュール追加」はヘッダー右側、閉じるボタンの左に配置する。
- レシピ詳細モーダル下部は「閉じる」「編集する」のみを残す。
- 食材管理の在庫一覧では、選択済み在庫がある場合だけ並び替え行の右側に「選択削除」を表示する。
- 買い物リストでは、選択済み買い物項目がある場合だけ並び替え行の右側に「購入済み」「選択削除」を表示する。
- 各チェックボックス、グループチェック、すべて選択の変更時に表示状態を更新する。

## 非対象

- 一括操作の処理内容変更
- 買い物リストや在庫の保存形式変更
- GAS / Spreadsheet 書き込み処理の変更
