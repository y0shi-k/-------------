# Summary

買い物リストを在庫リストと同じ操作感に揃え、出自を明細行ごとに保存する。画面では明細をそのまま見る「出自別」と、同じ品名+単位をまとめる「材料まとめ」を切り替えられるようにする。

## 仕様

- 買い物リストシートは既存6列に `出自種別`, `予定日`, `食事区分` を追加する。
- 既存データは追加列が空でも読み込める。
- 手動追加は `出自種別=手動追加` として保存する。
- レシピ詳細からの追加は `出自種別=レシピ詳細` として保存する。
- 献立由来は `出自種別=献立スケジュール`, `予定日`, `食事区分`, `紐づくレシピ名` を保存する。
- 材料まとめ表示では同じ `品名 + 単位` を1行に合算し、展開時に出自別明細を表示する。
- まとめ行チェックは配下明細をまとめて選択する。展開中は明細ごとの選択もできる。
- 削除は購入済ではなく完全削除として扱う。

## 非対象

- 数量単位の換算
- 買い物リスト以外のSpreadsheetスキーマ変更
- GASデプロイ作業
