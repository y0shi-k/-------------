# Summary

レシピ一覧カードでジャンルチップの文字が狭く省略されすぎる問題を直す。右側の3操作ボタンをタイトル行へ移し、2行目のジャンル表示領域を広げる。

## 仕様

- カード内の右操作ボタン群は、タイトル行の右端に配置する。
- タイトル行は `レシピ名 + 同期バッジ + 操作ボタン` の構成にする。
- 2行目は `材料/調理回数/登録日` とジャンル要約を同じ行に置き、ジャンル側が残り幅を使えるようにする。
- ジャンル要約はTKT-0062の最大3件 + `+N` を維持する。

## 非対象

- ジャンル保存形式変更
- レシピ一覧検索仕様変更
- GAS/Spreadsheet変更
