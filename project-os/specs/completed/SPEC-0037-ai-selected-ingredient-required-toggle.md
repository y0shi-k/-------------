# Summary

指定食材からAIレシピを考案する際、選んだ食材をすべて必須扱いにすると、使えたら使いたい食材まで強制される。選択済みバッジで必須/任意を切り替え、Geminiへのプロンプトに反映する。

## 仕様

- 選択済み食材は `required` 状態を持つ。
- 食材を新規選択した時点では `required: true` とする。
- 選択済みバッジ本体クリックで `required` を反転する。
- `×` クリックは選択解除のみを行う。
- `selected` モードのプロンプトは必須食材と任意食材を別見出しで出力する。

## 非対象

- AI生成結果の保存形式変更
- レシピ集シートや在庫シートのスキーマ変更
- GASデプロイ変更
