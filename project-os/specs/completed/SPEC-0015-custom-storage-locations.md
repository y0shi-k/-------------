# Summary

在庫管理画面の保存場所フィルタを固定タブからユーザー管理可能なチップに変更する。保存場所は新しいDB設定を持たず、在庫・登録待ちの `loc` 値とセッション内候補から導出する。

## 非対象

- 複数タグ機能
- 保存場所専用シートや列の追加
- 端末をまたぐ未使用候補の永続化
