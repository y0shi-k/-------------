# Summary

非同期で進む写真取得やGAS処理について、ユーザーが「今何をしているか」を把握できるようにする。全画面オーバーレイとは別に、操作を止めない軽量な下部ステータスを追加する。

## 仕様

- 料理履歴写真の未取得状態は、写真枠内にスピナーと「写真を取得中」を表示する
- 写真取得中は画面下部に「料理履歴の写真を読み込み中」などの処理名を表示する
- ステータス表示は画面操作をブロックしない
- 処理完了後は短時間だけ完了メッセージを表示して自動で隠す

## 非対象

- 新しい同期処理
- 画像取得件数の増加
- Drive共有設定変更
