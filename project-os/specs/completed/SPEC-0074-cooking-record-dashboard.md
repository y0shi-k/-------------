# Summary

モードCを「料理履歴のカード一覧」から、作った結果を振り返り、次の献立やレシピ改善につなげる料理記録ダッシュボードへ拡張する。

## 背景

モードA/Bの食材管理・献立レシピが整ってきた一方、モードCは記録の活用導線が弱い。レシピ管理・食事記録アプリの一般的な使い方に合わせ、カレンダー、タイムライン、振り返りの3層で記録を見せる。

## 仕様

- モードC通常表示の上部に記録サマリーを追加する。
- 料理履歴は `calendar`, `timeline`, `insights` の3ビューで表示する。
- `mealType` は保存列を増やさず、同日・同一レシピの献立スケジュールから推定する。見つからない場合は `その他` とする。
- `tags` は保存列を増やさず、レシピジャンル、写真有無、評価、メモ有無から表示用に派生する。
- カレンダーは月移動と日付選択を持ち、選択日の履歴カードを表示する。
- タイムラインは既存の履歴カードを日付グループ化し、フィルタを追加する。
- 振り返りは既存の `state.cookingHistory`, `state.recipes`, `state.schedule` から派生集計する。

## 非対象

- 料理履歴シートへの新規列追加
- AI分析、高度な統計、栄養/カロリー管理
- GASデプロイ、Spreadsheetの手動編集、Driveファイル操作
