# Summary

レシピ集一覧カードでジャンルチップが増えてカードが縦に膨らむ問題を防ぐ。ジャンルは2行目の登録日の右側へ移動し、多い場合は最大3件 + `+N` に要約する。編集画面ではジャンルチップのD&Dで表示優先順を調整できるようにする。

## 仕様

- 一覧カードの1行目は、レシピ名と同期バッジだけを表示する。
- 2行目は、既存の `材料 N 品目 | 調理回数 N | 登録 YYYY-MM-DD` の右側にジャンル要約を表示する。
- 一覧ジャンル要約は `genre` 配列順の先頭3件をチップ表示し、残りは `+N` チップにまとめる。
- `+N` チップの `title` には省略対象のジャンル名を `、` 区切りで入れる。
- 編集画面の選択済みジャンルチップはドラッグ可能にし、DOM順を `r-genres` hidden inputへ反映する。
- 候補のON/OFF、新規追加、削除後も hidden input の順序を正として再描画する。

## 非対象

- Spreadsheet列追加・GAS変更
- ジャンルの検索/保存仕様変更
- タッチイベント独自実装によるD&D完全対応
