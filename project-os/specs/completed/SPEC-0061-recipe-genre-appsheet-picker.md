# Summary

レシピ追加/編集モーダルのジャンル欄を、現在の「＋で1件ずつ追加するタグ入力」から、AppSheet風の検索付きチェック式複数選択UIへ変更する。

## 仕様

- `r-genres` は既存どおりJSON配列文字列を保持する。
- 候補は `DEFAULT_RECIPE_GENRES`、既存レシピのジャンル、現在選択中ジャンルから重複なしで生成する。
- ジャンル欄クリックまたは検索入力フォーカスでポップオーバーを開く。
- 候補一覧では選択済みジャンルを除外せず、チェック状態として表示する。
- 候補行クリックで選択状態をトグルし、ポップオーバーは開いたままにする。
- 検索文字列が候補に完全一致せず未選択の場合は、新規追加行を表示する。
- Enterは先頭候補のトグル、候補がなければ検索文字列の新規追加に使う。
- `×` は検索文字列があれば検索クリア、なければポップオーバーを閉じる。

## 非対象

- Spreadsheet列追加・GAS変更
- ジャンル候補専用シートや永続設定
- レシピ一覧の検索・表示仕様変更
