# Summary

調理中に手順文を読む時、Canvas版と同じように材料名と大さじ/小さじ分量を分けて読みやすくする。手順本文は変更せず、表示だけを変える。

## 仕様

- 対象は `web/src/components/recipe-meal-workspace.tsx` と `web/src/app/globals.css`。
- `chipifyStep` は `RecipeIngredient` を参照し、名前・分類から材料名チップを作る。
- 手順文中の `大さじ1` / `小さじ1` を検出し、赤/青の分量チップにする。
- 手順文に分量がない材料・調味料は、登録済み使用量を材料名の直後へ別チップで補う。
- 本文にすでに `大さじ1` / `小さじ1` が続く場合は、登録済み量を追加しない。
- チップの押下動作は従来どおり材料ハイライトに使う。
- 材料一覧側の分量も `大さじ1` / `小さじ1` の順に整える。

## 非対象

- AI生成プロンプトの変更
- 既存レシピ本文の自動補正
- 工程別消費量の管理
