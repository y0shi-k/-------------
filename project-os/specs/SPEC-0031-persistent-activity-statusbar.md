# Summary

TKT-0029で追加した下部ステータス表示を、一般的なアプリのステータスバーに近い常設1行領域へ変更する。処理中だけ浮かび上がるカード表示をやめ、最下部の固定領域に状態文言を表示することで、表示切替時のレイアウトズレを防ぐ。

## 仕様

- ステータスバーは画面最下部に固定し、通常時も領域を確保する
- ボトムナビはステータスバー領域の上に常時固定する
- メインコンテンツの下余白は、ボトムナビとステータスバー分を固定で確保する
- 手動一括同期は非ブロッキング通信として扱い、常設ステータスバーで進行状況を表示する
- 手動一括同期中に再度同期要求された場合は、同期を増やさずトーストで通知する

## 非対象

- GAS payload の書き込み内容変更
- Spreadsheet / Drive のスキーマ変更
- AI処理や画像解析のブロッキング挙動変更
