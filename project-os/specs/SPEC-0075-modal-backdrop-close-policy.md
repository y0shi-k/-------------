# Summary

モーダル背景クリックによる意図しない入力破棄を防ぐため、外側クリックで閉じる対象を明示的に制御する。

## 仕様

- 画面外クリックで閉じないモーダル:
  - 食材の追加/編集、保存場所の管理、買い物追加
  - レシピ追加/編集、テキストからレシピ追加、AIレシピリクエスト
  - 買い物候補選択、献立レシピ選択、スケジュール追加
  - 消費量調整、料理の記録
- 画面外クリックで閉じるモーダル:
  - AI考案メニュー、献立スロット操作メニュー
  - 献立削除確認、献立一括削除確認、代替品選択
- AIレシピプレビューは `state._aiPreviewMode` により、`generated` では閉じず、`viewing` では閉じる。

## 非対象

- モーダル内のフォーム内容、保存処理、同期処理の変更
- GAS側の変更
- キーボードショートカット追加
