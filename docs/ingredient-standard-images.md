# 食材標準画像カタログ

TKT-0175で追加した標準画像は、外部素材を使わず、このリポジトリ向けに作成したSVGアイコンです。

## 利用条件

- 出典: Codexによる自作SVG
- 第三者素材: なし
- ライセンス条件: 外部素材由来の追加条件なし。プロジェクト内アセットとして利用する
- 加工: プロジェクト内で色・形・ファイル名の変更可

## 対応範囲

`web/src/lib/ui/ingredient-image.ts` のresolverで、主要食材や調味料の表記ゆれを標準アイコンに寄せています。

- 肉・魚: 鶏肉、豚肉、牛肉、ひき肉、鮭など
- 野菜: 玉ねぎ、にんじん、じゃがいも、キャベツ、トマト、きのこ、小松菜、大根、長ねぎなど
- 卵・乳製品: 卵、牛乳、チーズ
- 主食: 米、パン、パスタ、うどん
- その他: 納豆、お茶
- 調味料: 醤油、味噌、塩、砂糖、酢、みりん、酒、油、カレールー

標準画像が見つからない食材は、既存の絵文字フォールバックに戻します。
